Codegen is an open-source AI development assistant, built on our powerful `CodeAgent` framework. It seamlessly integrates with your existing development tools - Slack for communication, GitHub for code management, and Linear for project tracking - to provide an intelligent coding companion right where you work.


This guide will help you set up and configure our AI-powered development assistant with Slack, GitHub, and Linear integrations. Follow these steps to get started quickly.
